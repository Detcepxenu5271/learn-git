# Git使用教程


## 原理
版本控制系统（version control system，VCS）
* 本地
* 集中化
* 分布式
	- 本地

Git 的文件结构
* .git 文件夹

三种状态
* 已提交
* 已修改
* 已暂存

git 安装
* [Download Git for Windows](https://git-scm.com/download/win)
* 设置用户名和邮箱（用途：记录信息）
	- `git config --global user.name "John Doe"`
	- `git config --global user.email johndoe@example.com`
